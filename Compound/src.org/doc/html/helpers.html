helpers¶

class pyeuv.helpers.LoadFile(f)[source]¶

Loads content of given filename. Decompresses if gzip or LZMA compression is detected Acts as context manager, so must be called using a with statement. Will take care of opening/closing files.

Example

with LoadFile('somefile.xz') as file_handle:
    contents = file_handle.read()
    do_something_with(contents)
