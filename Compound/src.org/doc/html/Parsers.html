Dat parser¶

pyeuv.Parsers.dat_parser.ls_dat(fn, datv1_metadatafile=None, verbose=False)[source]¶

Lists signals available in the given dat file.

Parameters

  • fn (str) – Full path to the dat file

  • datv1_metadatafile (str) – Required only when parsing datv1 data.

  • verbose (bool) – Print messages from the parser module

pyeuv.Parsers.dat_parser.parse_dat(filenames, signal_list, datv1_metadatfile=None, verbose=False)[source]¶

Processes input dat files into pandas dataframes.

Parameters

  • filenames – List of files (full path) to process

  • signal_list – List of signals to extract For example: BDeuvEnergyAvgOn from RT5

  • datv1_metadatafile – Path to metadatafile, for DATv1. Will try to autodetect if set to None (default)

  • verbose – If true, prints filenames being parsed

Returns

pandas.DataFrame: Dataframe

pyeuv.Parsers.dat_parser.parse_dat_directory(directory, rt_type, signal_list=None)[source]¶

Parse RT files of the given type in this directory. Unless a signal_list is provided it will parse all available signals in the RT file. Can only process one type of RT data at the same time.

Parameters

  • directory (str) – Directory containing the RT files

  • rt_type (str) – Type of RT to parse. (RT05, RT05 etc…)

  • signal_list (list<str>, optional) – Signals in the RT file to parse.

Returns

parsed data, or None if the directory is empty

Return type

pandas.DataFrame or None

pyeuv.Parsers.dat_parser.parse_dat_directory_gui(rt_type, signal_list=None)[source]¶

Parse RT files of the given in the directory selected by the user. Upon calling this function a tk file selection diaglog is opened. Unless a signal_list is provided it will parse all available signals in the RT file. Can only process one type of RT data at the same time.

Parameters

  • rt_type (str) – Type of RT to parse. (RT05, RT05 etc…)

  • signal_list (list<str>, optional) – Signals in the RT file to parse.

Returns

parsed data, or None if the directory is empty

Return type

pandas.DataFrame or None
