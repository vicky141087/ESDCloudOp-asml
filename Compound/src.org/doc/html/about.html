About pyeuv¶

PyEUV - Python modules for working with EUV data.
Maintainer: Rob de Laat (SW DE DATA Data Science & Engineering) <rob.de.laat@asml.com>
Copyright ASML 2020
Version: 1.2

This module provides parsers and interfaces to work with EUV Data. This version contains the following Features:

  1. Interface to InfluxDB/Grafana (EUV Dashboard)

  2. Interface to PMA NG

  3. DAT parser, DATv0, DATv1 and DATv2 are supported, (includes compressed DAT)

  4. Parsers for source KPI and scanner Parameters

  5. NXE Source Collector data analysis algorithms

  6. NXE Illuminator data analysis algorithms

  7. NXE Source Tin Management analysis algorithms

The parsers and interfaces output pandas and numpy objects that allow for easy data manipulation, making this module perfect for use in your, Python tools, Jupyter Notebooks and hack scripts.

Please use this module responsibly. Persistent abusers will be tracked down for a friendly conversation.

Before asking questions, have a look at the examples folder, the inline documentation of the module and the Q&A.
